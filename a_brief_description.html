A brief description of HTML

  • According to the definition

    HTML stands for Hypertext Markup Language. HTML is the language used to create webpages. Hypertext refers to the hyperlinks that an HTML page may contain. Markup language is the way tags are used to define the page layout and elements within the page.

  • As for another meaning

    HTML is a markup language that web browsers use to interpret and compose text, images, and other things into visual or audible web pages.

  • Structure of HTML5

    When working with HTML, we use simple code structures tag and attributes to mark up a website page. For example, we can create a paragraph by placing the enclosed text within a starting <p> and closing </p> tag.
  • The three block level tags every HTML document needs to contain are:
    1. Html
    2. Head
    3. Body
