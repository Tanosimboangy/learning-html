List of all elements in HTML5

Definition of an HTML element

  1. The <html></html> tag is the highest level element that encloses every HTML page.
  2. The <html> tag tells the browser that this is an HTML document.
  3. The <html> tag represents the root of an HTML document.
  4. The <html> tag is the container for all other HTML elements (except for the <!DOCTYPE> tag).
  5. The <html> element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.
