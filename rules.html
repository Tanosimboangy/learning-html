List of HTML rules

These are some rules for coding or writing HTML :

  • An HTML document must start with <!DOCTYPE html>.
  • You still have to specify a language: <html lang=“en”>.
  • You have to define a character encoding: <meta charset=“utf-8”>.
  • You actually have to include a viewport: <meta name=”viewport” content=”width=device-width, user-scalable=no”>.
  • Tags are always surrounded by angle brackets (less-than/greater-than characters) as in <Head>.
  • You can't have and end tag without starting tag.
  • Every start tag needs: start tag a less-than character and end with a greater-than character.
  • Every end tag needs: start with a less-than character and end with a greater-than character followed by slash.
  • Filenames must be lower case.
  • If you need your filenames to be longer than one word, they should be separated by hyphens.
  • You have to nest the element properly.
  • you have to indent every child element.
  • To start a new paragraph, use <p> at the beginning of the paragraph and </p> at the end of the paragraph.
