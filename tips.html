Tips for writing HTML page

  • Some ways to write HTML quicker:

    • On the visual studio code, use the empty code from the template as a sample, so that you don't need to create anymore. Only if you have a template.
    • Write your code in order, from the head until to body.
    • First and formost, use some of the useful keyboard shortcuts, like ctrl+c to copy and ctrl+v to paste.
    • You should be able to type the keyboard as fast as you can. Practice to improve your typing.
    • Don't rely on using the mouse all the time. It slows down your speed.
    • Use tab to indent the child element instead of typing the white space many times.
    • Use ←enter to close the closing tag, or to rewrite the word already existed.

      • For example:

        <section>
        <h2>
        learning html
        </h2>
        </sec ←use ←enter to close this tag rather than typing the left content.
    • Use <shift↑>+<alt>+f to format the document.
    • Apart from that, you need to remember some necessary HTML elements and understand their fonction.

  • Some tools to use:

    • All keyboard shortcuts like shift+tab.
    • Visual Studio Code is required to write the document.
    • All HTML elements.
    • The Validator page.

  • To make sure that your work is high-quality:

    • First, apply perfectly all of the rules for HTML elements.
    • Use tab to indent every child element, so that you can avoid the misindenting. Indent all element correctly.
    • Don't forget to close first the last tag you open.
    • Pay attention on grammar and spelling. If possible, use Word Document to write your document. you can avoid the mistakes in instant.
    • Use Validator to check the whole work. So pass your code to Validator.
